# rakhadjo.me personal website

See the tech I use to implement my personal website.

### Branches

- `master` used as the production branch, for deploys on github pages.
- `source` used for adding new features, previewing. This is the feature branch.

### Tech Implemented

- `gh-pages` for website hosting
- `gatsby.js` for react-based front-end framework
- `javascript` and `jsx` for writing data entry and components.
- `SCSS`for styling
